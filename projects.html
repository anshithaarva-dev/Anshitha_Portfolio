Data Analysis & Visualization

iPhone 16 Review Analysis Based on YouTube Comments

Analyzed public reception of the iPhone 16 Pro by scraping and preprocessing YouTube comments using Python. Applied VADER sentiment analysis to classify viewer sentiment and used topic modeling (LDA) and clustering (Sentence-BERT) to identify key discussion themes. Visualized results with Tableau to uncover consumer sentiment trends and product perception insights. Really interesting to see how people actually felt about the new features!

Global Tourism Analysis

Working with UN Tourism and the World Travel & Tourism Council datasets, I analyzed and explored global tourism patterns. Used Python for data cleansing and visualization, and also forecasted post-pandemic recovery trajectories using time-series forecasting models. Found some really interesting trends like the recovery of international travel, growing interest in sustainable tourism, and how digitization is changing traveler behavior. Created interactive dashboards to present findings for strategic tourism industry planning.
